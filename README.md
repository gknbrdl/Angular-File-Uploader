# Angular File Uploader

## First Step

Inject your angular project

```
angular.module('app',['FileUploader']);

```

## Last Step

```
<uploader ng-model="" upload-url="your_upload_request_url"></uploader>

```

# Required Types

* ng-model
* upload-url

## Custom Types

 * file-type (jpg,pdf,gif,png)
 * file-size (5000) 5mb
 * upload-button (true/false)
